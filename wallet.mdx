---

## Autenticação

Esta requisição requer autenticação via headers customizados:

<ParamField header="stpi" type="string" required>
  Seu Client ID.
</ParamField>

<ParamField header="stps" type="string" required>
  Seu Client Secret.
</ParamField>

## Resposta

<ResponseField name="statusCode" type="number">
  Código de status HTTP da requisição. Exemplo: `200`.
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva da operação. Exemplo: `"Saldo consultado com sucesso."`.
</ResponseField>

<ResponseField name="data" type="object">
  Objeto contendo os dados do saldo disponível.

  <Expandable title="propriedades">
    <ResponseField name="available_cents" type="number">
      Saldo disponível para uso imediato ou saque, retornado **em centavos**.

      Exemplo: `120000` representa `R$ 1.200,00`.

      **Recomendação:** utilize este campo para cálculos financeiros e divida por `100` apenas para exibição no front-end.
    </ResponseField>

    <ResponseField name="currency" type="string">
      Moeda referente ao saldo da carteira. Padrão: `"BRL"`.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  Data e hora da consulta no formato ISO 8601.
</ResponseField>

<RequestExample>

```bash cURL
curl --location 'https://api.stylepay.com.br/api/v1/wallet/balance' \
  --header 'stpi: seu_client_id' \
  --header 'stps: seu_client_secret'
```

```javascript JavaScript
const response = await fetch('https://api.stylepay.com.br/api/v1/wallet/balance', {
  method: 'GET',
  headers: {
    stpi: 'seu_client_id',
    stps: 'seu_client_secret'
  }
});

const json = await response.json();

const saldoReais = json.data.available_cents / 100;

console.log(`Saldo disponível: R$ ${saldoReais.toFixed(2)}`);
```

```python Python
import requests

url = "https://api.stylepay.com.br/api/v1/wallet/balance"

headers = {
    "stpi": "seu_client_id",
    "stps": "seu_client_secret"
}

response = requests.get(url, headers=headers)
data = response.json()

saldo_reais = data["data"]["available_cents"] / 100

print(f"Saldo disponível: R$ {saldo_reais:.2f}")
```

```php PHP
<?php

$url = "https://api.stylepay.com.br/api/v1/wallet/balance";

$headers = [
    "stpi: seu_client_id",
    "stps: seu_client_secret"
];

$ch = curl_init($url);

curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);

curl_close($ch);

$json = json_decode($response, true);

$saldo_reais = $json["data"]["available_cents"] / 100;

echo "Saldo disponível: R$ " . number_format($saldo_reais, 2, ",", ".");

?>
```
