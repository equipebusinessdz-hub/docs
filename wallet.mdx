---
title: "Consultar Saldo"
api: "GET https://api.stylepay.com.br/api/v1/wallet/balance"
description: "Consulta o saldo disponível na carteira correspondente (disponível para saque e movimentações)"
---

## Autenticação

Esta requisição requer autenticação via headers customizados:

<ParamField header="stpi" type="string" required>
  Seu Client ID
</ParamField>

<ParamField header="stps" type="string" required>
  Seu Client Secret
</ParamField>

## Resposta

<ResponseField name="statusCode" type="number">
  Código de status HTTP da requisição (ex: 200).
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva (ex: "Saldo consultado com sucesso.").
</ResponseField>

<ResponseField name="data" type="object">
  Objeto contendo os dados do saldo.
  <Expandable title="propriedades">
    <ResponseField name="available_cents" type="number">
      Saldo disponível para uso imediato ou saque **em centavos** (inteiro).  
      **Dica:** Sempre divida este valor por 100 no front-end para exibir o valor em Reais (R$).
    </ResponseField>
    <ResponseField name="currency" type="string">
      Moeda referente ao saldo da carteira (padrão: "BRL").
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  Data e hora (ISO 8601) da consulta do saldo.
</ResponseField>

<RequestExample>

```bash cURL
curl --location 'https://api.stylepay.com.br/api/v1/wallet/balance' \
--header 'stpi: seu_client_id' \
--header 'stps: seu_client_secret'
