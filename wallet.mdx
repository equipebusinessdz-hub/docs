## Autenticação
Esta requisição requer autenticação via headers customizados:
<ParamField header="stpi" type="string" required>
  Seu Client ID.
</ParamField>
<ParamField header="stps" type="string" required>
  Seu Client Secret.
</ParamField>
## Resposta
<ResponseField name="statusCode" type="number">
  Código de status HTTP da requisição. Exemplo: `200`.
</ResponseField>
<ResponseField name="message" type="string">
  Mensagem descritiva da operação. Exemplo: `"Saldo consultado com sucesso."`.
</ResponseField>
<ResponseField name="data" type="object">
  Objeto contendo os dados do saldo disponível.
  <Expandable title="propriedades">
    <ResponseField name="available_cents" type="number">
      Saldo disponível para uso imediato ou saque, retornado **em centavos**.
      Exemplo: `120000` representa `R$ 1.200,00`.
      **Recomendação:** utilize este campo para cálculos financeiros e divida por `100` apenas para exibição no front-end.
    </ResponseField>
    <ResponseField name="currency" type="string">
      Moeda referente ao saldo da carteira. Padrão: `"BRL"`.
    </ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="timestamp" type="string">
  Data e hora da consulta no formato ISO 8601.
</ResponseField>

```go
package main
import (
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)
type BalanceData struct {
    AvailableCents int `json:"available_cents"`
}
type BalanceResponse struct {
    Data BalanceData `json:"data"`
}
func main() {
    url := "https://api.stylepay.com.br/api/v1/wallet/balance"
    
    req, _ := http.NewRequest("GET", url, nil)
    req.Header.Set("stpi", "seu_client_id")
    req.Header.Set("stps", "seu_client_secret")
    
    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()
    
    body, _ := io.ReadAll(resp.Body)
    
    var responseData BalanceResponse
    json.Unmarshal(body, &responseData)
    
    saldoReais := float64(responseData.Data.AvailableCents) / 100.0
    fmt.Printf("Saldo disponível: R$ %.2f\n", saldoReais)
}
```
