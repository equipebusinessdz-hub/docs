## Exemplo Rápido

Veja como é simples gerar um QR Code PIX:

<CodeGroup>

```javascript Node.js
const response = await fetch('https://api.stylepay.com.br/api/v1/gateway/request-qrcode', {
  method: 'POST',
  headers: {
    'stpi': 'seu_client_id',
    'stps': 'seu_client_secret',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: 100.50,
    external_id: "pedido_123",
    payer: {
      name: "João Silva",
      document: "12345678900",
      email: "joao@email.com"
    },
    postbackUrl: "https://seusite.com/webhook"
  })
});

const { qrcode, qrcode_image } = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.stylepay.com.br/api/v1/gateway/request-qrcode',
    headers={
        'stpi': 'seu_client_id',
        'stps': 'seu_client_secret',
        'Content-Type': 'application/json'
    },
    json={
        'amount': 100.50,
        'external_id': 'pedido_123',
        'payer': {
            'name': 'João Silva',
            'document': '12345678900',
            'email': 'joao@email.com'
        },
        'postbackUrl': 'https://seusite.com/webhook'
    }
)

data = response.json()
qrcode = data['qrcode']
qrcode_image = data['qrcode_image']
```

```php PHP
<?php

$response = file_get_contents('https://api.stylepay.com.br/api/v1/gateway/request-qrcode', false, stream_context_create([
    'http' => [
        'method' => 'POST',
        'header' => [
            'stpi: seu_client_id',
            'stps: seu_client_secret',
            'Content-Type: application/json'
        ],
        'content' => json_encode([
            'amount' => 100.50,
            'external_id' => 'pedido_123',
            'payer' => [
                'name' => 'João Silva',
                'document' => '12345678900',
                'email' => 'joao@email.com'
            ],
            'postbackUrl' => 'https://seusite.com/webhook'
        ])
    ]
]));

$data = json_decode($response, true);
$qrcode = $data['qrcode'];
$qrcode_image = $data['qrcode_image'];
?>
```
