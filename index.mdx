# StylePay API

A StylePay API permite que plataformas recebam e enviem pagamentos PIX de forma simples e segura, com controle de saldo em carteira, consulta de transações e webhooks em tempo real.

## Casos de uso

- **Receber pagamentos PIX (cash-in)** via QR Code dinâmico.
- **Realizar pagamentos PIX (cash-out)** para seus clientes.
- **Consultar saldo de carteira** e **transações**.
- **Receber notificações via webhook** de atualização de status.

## Ambientes

> Substitua pelos seus domínios reais.

- **Produção**  
  `https://api.seu-dominio.com`

## Autenticação

A autenticação é feita por **chaves de integração** enviadas nos headers:

- `stpi`: Client ID
- `stps`: Client Secret

Esses headers são obrigatórios em todos os endpoints sob `/api/v1/gateway` e `/api/v1/wallet`.

## Fluxo básico

1. Criar uma cobrança PIX (`request-qrcode`).
2. Esperar o webhook de confirmação de pagamento.
3. Opcionalmente, consultar o status da transação.
4. Realizar saques PIX (`pix-payment`) quando necessário.
